JAM development tooling. Code, build, deploy and test your multi-service projects.

## Documentation

Documentation lives in `docs/` and is powered by [mdBook](https://github.com/rust-lang/mdBook). To preview locally install `mdbook` (`cargo install mdbook`) and run:

```bash
mdbook serve docs --open
```
